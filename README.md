# 🔥 AI CV Optimiser

An AI-powered CV optimisation tool that helps users improve their CVs by analysing them against job descriptions, providing ATS scoring, and suggesting keyword improvements.

---

## 🚀 Live Demo

Try it here:  
👉 https://ai-cv-optimiser-delta.vercel.app/

---

## 📌 Overview

This project is an MVP designed to simulate real-world applicant tracking system (ATS) analysis. It uses AI to:

- Score CVs based on job relevance
- Identify missing keywords
- Suggest improvements to increase interview chances
- Provide structured feedback for CV optimisation

---

## ⚙️ Features

- 📊 ATS-style CV scoring
- 🧠 AI-powered feedback analysis
- 🔍 Keyword matching against job descriptions
- ✍️ CV improvement suggestions
- 🌐 Fully deployed web app via Vercel
